MediaTek LinkIt™ Development Platform for RTOS

This platform, based on FreeRTOS with additional open-soure modules, offers a common toolset and API for multiple chipsets and hardware. You can create Wi-Fi connected smart home devices using the LinkIt 7687 HDK by SAC, or power efficient Wearables with GNSS and Bluetooth using the LinkIt 2523 HDK by SAC. The platform is ARM Cortex-M4 based with development and debugging support in Keil μVision, IAR Embedded Workbench and GCC.

MediaTek LinkIt™ Smart 7688

Rapidly create Wi-Fi connected smart home products or other smart IoT devices based on the MediaTek MT7688AN powered by OpenWrt Linux. Create applications in Python, Node.js or native C/C++, or use the Arduino IDE to create apps for the MCU included on the LinkIt Smart 7688 Duo. This platform is ideal for mobile and web developers who want to migrate over to IoT.

Tinitell

Tinitell is a wristphone for kids. Kids can communicate with parents or guardians with a simple press of a button. The wristphone has an integrated GPS function that enables parents to locate their kids using a map in the companion Tinitell mobile app.

The Hug Smartwatch

The Hug Smartwatch is designed to help users control their gadgets with just the motion of a hand. Hug is the world’s first Gesture Control Smartwatch with integration that enhances the customer experience in Virtual Reality with the Oculus Rift. Developed using the MediaTek LinkIt™ ONE platform, based on MT2502, the Hug Smartwatch enables its users to make calls, track physical activity and nutrition, play music and find misplaced or stolen devices.
