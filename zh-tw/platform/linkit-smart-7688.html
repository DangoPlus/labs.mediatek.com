LinkIt™ Smart 7688

概述

LinkIt Smart 7688 開發版基於聯發科技的 MT7688AN 系統級晶片，提供高效能的 802.11n 1T1R Wi-Fi 和各種周邊連接能力。軟體開發環境基於 OpenWrt 作業系統，預設安裝了許多物聯網裝置開發常用的套件，包含了多種程式語言的支援，例如 Python、Node.js 和 C 語言。 LinkIt Smart 7688 Duo 開發版則額外搭配了 ATmega32U4 微控制器，可使用 Arduino 開發環境連接更多周邊。

開發板

LinkIt Smart 7688 開發平提供硬體開發包 (HDKs) 讓您實現物聯網概念。該平台提供兩個 HDK。

LinkIt Smart 7688

使用 MT7688AN 作為微處理器 (MPU) 。 支援 Python、Node.js 和 C 語言開發。

功能特點

  • 支援 Wi-Fi、USB host 和 SD 卡

  • 接腳提供的規格包含： PWM、I2C、SPI、I2S 和 UART (Lite)

  • 開源的硬體參考設計：包含原理圖、布局圖、和接腳細節

LinkIt Smart 7688 Duo

LinkIt Smart 7688 Duo開發板使用 MT7688AN 搭配 ATmega32U4 微控制器 (MCU) 。


除了支援 Python、Node.js 和 C 語言之外，也支援以 Arduino 開發環境進行 MCU 端的應用程式開發。

功能特點

  • 支援 Wi-Fi, USB host 和 SD 卡

  • MT7688AN接腳：Ethernet、USB Host 和 UART

  • MCU 接腳: ADC、PWM、I2C、SPI 和 UART

  • 開源的硬體參考設計：包含原理圖、布局圖、和接腳細節

MT7688AN

提供高效能與低功耗的智慧居家和辦公室裝置解決方案。


MT7688AN 系統級晶片是該開發平台的核心，也是家電自動化的橋接中心。

MT7688 Datasheet

功能特點

  • 高效能802.11n 1T1R Wi-Fi

  • 支援多達 256MB 的 RAM 及外接 SD卡

  • Wi-Fi 加密引擎

  • 各種周邊連接能力
