MediaTek Cloud Sandbox

概述

在設計與開發可穿戴和物聯網設備原型時，您可能希望能收集，排序和視覺化開發原型所收集到的數據。您或許也想透過遠程控制您的設備原型，並且提供這些功能給一同協作的開發者與測試人員。

現在您無需花費過多的時間與金錢來尋找或購買這類雲端服務，MediaTek Cloud Sandbox（MCS）為您提供免費的服務，您可以使用它快速的打造出您所構想的雲端服務的原型。


您可以在我們提供的強大的網頁儀表板介面上，看到使用我們提供的 RESTful API 從裝置蒐集回來的圖形化的資料。之後，您可以通過從儀表板發出命令控制您的穿戴式和物聯網裝置。此外，我們提供附屬的手機應用程式讓您能查看蒐集到的數據，並從任何地方控制您的裝置。


MCS 目前提供服務給全球的會員，並且在中國內部架設有獨立的資料中心，用以提供最佳的連線品質與服務。

重點功能

  • 定義專屬於您的穿戴式和物聯網產品原型以及個人化裝置。

  • 定義各種類型的資料點。例如：GPS 座標，溫度，濕度。

  • 在同個產品原型下建立多個裝置。

  • 透過 Restful API 隨時在裝置和 MCS 平台間推播各式資料點。

  • 遠端控制您的裝置。例如：遠程開關設定。

  • 圖形化的資料點介面。

  • 當資料點超過特定觸發條件預設範圍時，發送通知。

  • 使用專屬的手機應用程式遠端控制裝置。

  • 建立產品原型，裝置，和蒐集來的資料報表。

  • 空中（Over the air）韌體更新。

  • 使用者權限管理。

  • 完整的教學文件。
