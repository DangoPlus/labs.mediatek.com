概述

隨著具有無線功能的家電用品日益增加的趨勢，對於許多智慧家電裝置的開發者來說，聯發科技 MT7681 所提供的從遠端控制裝置的能力 – 例如開啟枱燈，調整空調溫度設定或解除大門鎖定等等，正是此平台的最佳用途與優勢。


針對物聯網裝置原型設計，我們提供了 LinkIt Connect 7681 開發板. 該開發板由一個 MT7681 模組為核心，加上 micro-USB 介面並提供MT7681晶片的所有I / O介面引腳. 這能使您快速的連線外部硬體周邊來建立裝置原型。 MT7681 模組的大小只有15x18mm，此設計讓它能夠輕易的整合至最終物聯網裝置的PCB上。

Key features

  • 支援 Wi-Fi STA 模式: 802.11b/g/n

  • 支援 Wi-Fi AP 模式: 802.11b/g

晶片

MT7681

支援 TCP/ IP 協定的嵌入式 Wi-Fi 物聯網系統單晶片。


聯發科技 MT7681 是個高整合的無線網路系統級晶片（SoC），並擁有物聯網裝置所需的嵌入式TCP/ IP協定堆疊。IoT 裝置經由 MT7681 將可以連線到其他智慧裝置或雲端應用和服務。


在 STA 模式時，MT7681 連線至無線 AP，之後可將與網站服務或雲端伺服器進行通訊。一個典型的用途是，使用者可透過智慧家庭的網站來遠端控制家裡的冷氣機。

SDK

聯發科技 LinkIt™ Connect 7681 SDK ，是基於 Andes 平台的軟體工具開發包。它能讓您建立支持 Wi-Fi 網路的物聯網裝置韌體。該 SDK 提供韌體更新工具，以及LinkIt Connect 7681 開發板軟體上傳工具。

功能特點

  • LinkIt Connect 7681 API 庫包含Smart Connection 和空中韌體更新（FOTA）

  • 基於 C 語言

  • 基於 Andes 平台的命令列編譯器

  • 提供 Wi-Fi 與網際網路 API、周邊介面 API，以及安全加解密 API

  • 支援 Wi-Fi AP 和 STA 模式. 支援 Wi-Fi 安全協定，包含WEP, WPA 和 WPA2-PSK。網路協定則包含 TCP/IP, DHCP, SNTP 和 DNS client

  • 支援微軟Windows（需安裝 Cygwin 環境）和 Ubuntu Linux
