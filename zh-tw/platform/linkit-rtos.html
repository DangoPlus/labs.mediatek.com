LinkIt™ 開發平台 for RTOS

概述

聯發科技 LinkIt™ 開發平台 for RTOS 是一個具備豐富功能的物聯網與穿戴式開發平台。該平台支援一系列聯發科技晶片，包含用於物聯網裝置的MT7687F, MT7697 and MT7697D，以及用於穿戴式裝置的MT2523D 和 MT2523G。本開發平台基於受歡迎的FreeRTOS作業系統，透過提供統一的程式介面與開發工具，您將能在多樣的系統單晶片上開發廣泛的消費性或商務物聯網裝置。

連網系統單晶片

一系列多樣化的系統單晶片（SoC），整合了無線通訊功能與微控制器，具有優秀功耗效能表現。


  • Wi-Fi 無線網路 (MT7687 family)
  • Wi-Fi 無線網路與Bluetooth LE低功耗藍牙 (MT7697 family)
  • 衛星定位與雙模式藍牙 Bluetooth EDR/LE(MT2523 family)

廣受歡迎的軟硬體架構

基於流行的ARM Cortex-M4硬體架構以及FreeRTOS作業系統，您可整合並利用各種多樣化的第三方工具，在LinkIt™ 開發平台 for RTOS上面進行開發。

豐富的軟體功能

本平台提供LinkIt SDK v4，整合了多種中介軟體，可用於快速開發廣泛類型的應用。軟體開發套件提供的功能有：TCP/IP，SSL/TLS，MQTT，Bluetooth EDR/LE profiles，感測器子系統，電池管理，以及遠端無線韌體更新。

LinkIt 7697

LinkIt 7697 是一塊針對物聯網應用的開發板。它基於 MT7697 系統單晶片，具有含浮點運算的 ARM Cortex-M4 微控制器，並整合了 802.11b/g/n Wi-Fi 無線網路與 Bluetooth 4.2 低功耗藍牙。

功能特色

  • 基於ARM Cortex-M4 的 MT7697 系統單晶片，時脈 192MHz

  • 802.11b/g/n 無線網路支持

  • Bluetooth 4.2 低功耗藍牙

  • 352 KB RAM

  • 4 MB 外接快閃記憶體

  • Micro-USB 接頭用於供電

  • 透過版載的 CP2102N USB 轉 UART 晶片提供 COM 串行口，用於燒錄

  • 提供系統重置按鈕與使用者自定義按鈕

  • 提供下列周邊介面：GPIO、UART、I2C、SPI、PWM、EINT、ADC、IrDA 與 I2S

  • 提供 SWD 除錯介面排針

  • 使用 LinkIt SDK 開發，可支援 GCC、IAR Embedded Workbench 以及 Keil uVision IDE（使用指南）

  • 提供額外擴充包以支援 Arduino IDE

  • 尺寸：48 x 26 mm

LinkIt 7687 HDK by SAC

LinkIt 7687 HDK （由品佳集團提供）是一塊針對物聯網應用的開發板，可用於設計與評估物聯網裝置。它基於MT7687F系統單晶片，在QFN68的封裝中，具有含浮點運算的 ARM Cortex-M4微控制器，並整合了802.11b/g/n Wi-Fi 無線網路功能。開發板提供豐富的周邊連接能力，能在連接至網路雲端服務的情況下，即時控制各種周邊硬體。

本開發板搭配LinkIt SDK v4，含有多樣的網路連線功能，能協助您整合各種周邊硬體至各種新類型的連網服務。

功能特色

  • 基於ARM Cortex-M4 的 MT7687F 系統單晶片

  • 802.11b/g/n 無線網路支持，具備內建天線

  • U.FL 接頭可用於外接天線或進行測試

  • Micro-USB接頭可用於供電，並提供串口與CMSIS-DAP介面，可進行開發與除錯

  • 接受外部 1.8 ~ 3.2V 電源，用 兩個AA電池就能驅動

  • 系統重置、RTC中斷與外部中斷，共三個按鈕

  • 提供電流功耗量測腳位

  • 相容於Arduino Uno (revision 3)的腳位配置，並提供額外八個擴充腳位

  • 尺寸： 6.05 x 10.85 公分

LinkIt 2523 HDK by SAC

LinkIt 2523 HDK （由品佳集團提供）提供您開發穿戴式裝置原型所需的一切功能。它使用聯發科技MT2523G，提供具備浮點運算能力的ARM Cortex-M4微控制器，並整合Bluetooth EDR/LE，以及衛星定位系統。開發板上並提供觸控顯示屏幕，以及為電池與感測器保留的擴充接頭。

功能特色

  • 基於ARM Cortex-M4的MT2523G 系統單晶片

  • 支持Bluetooth EDR/LE與衛星定位系統

  • 內建天線，並可透過 SMA 接頭外接天線

  • Micro-USB接頭可用於供電，並提供串口與CMSIS-DAP介面，可進行開發與除錯

  • 具備320 x 320 像素的觸控顯示屏

  • 喇叭與耳麥插孔

  • 4GB eMMC 快閃記憶體儲存媒體

  • SD卡插槽

  • 電池、感測器與相機的擴充接腳

  • 電流功耗量測接腳

  • 尺寸： 7 x 21.5 公分

SDK

聯發科技 LinkIt™ 軟體開發包 v4 提供您開發應用所需要的軟體與工具。本軟體架構切分為三部分，由BSP（開發板支援軟體包），中介軟體以及應用層所組成。


開發板支援軟體包當中包含了系統單晶片的硬體驅動程式，以及 FreeRTOS 作業系統的整合。此層在不同的晶片之間提供了共通的硬體抽象層（HAL），並提供開發板上內附的周邊硬體的驅動程式，如顯示與觸控。


中介軟體層提供了應用程式開發所需要的重要功能模組，如Wi-Fi與Bluetooth的協議堆疊，IP實做（lwIP）與其他網際網路協議，以及電池管理、感測器子系統，遠端無線韌體更新等功能。


應用層則提供了易於使用的範例程式，參考設計，以及示範專案，使開發者可以快速將之利用於本身的開發案中。


您可以選擇使用免費的GCC工具鍊開發，或使用專業的整合開發環境，如 Keil μVision 和 IAR Embedded Workbench 進行開發。


軟體開發環境

  • ARM Keil μVision 擴充包 （適用於Windows）
  • IAR Embedded Workbench（適用於Windows）
  • GCC （適用於Linux，或安裝MinGW的Windows）
Keil μVision 擴充包
