LinkIt™ Assist 2502

API 指南
開發資源
討論區 FAQ

概述

聯發科技LinkIt™ Assist 2502 讓開發者在專業的開發環境下進行穿戴式與物聯網（IoT）裝置的硬體和軟體原型開發。


本開發平台是基於現今最小的穿戴式系統級晶片組 (SoC), 聯發科技 MT2502 （Aster）。此SOC 也能夠和聯發科技節能的 Wi-F i與 GNSS 晶片組一同運作，而這意味著您現在就可以很輕鬆的建立裝置並與其他智慧裝置或雲端服務連線。

LLinkIt Assist 2502開發平台是由 MT2502 (Aster) 和配套 GNSS 與 Wi-Fi 晶片組, 還有觸控面板 LCM CTP，Memory LCD 和OLED 螢幕（可更換）所構成的 LinkIt Assist 2502 開發平台，它的韌體是以 RTOS 為核心。在核心之上運行著各種驅動程式、仲介軟體與通訊協定。Runtime執行環境提供了以C語言為基礎的API來存取系統功能，讓您使用LinkIt Assist 2502 軟體開發工具包 SDK 2.0 建立應用。

功能特點

  • 基於 C 語言的 API，讓您存取和控制 MT2502 SoC 的硬體功能和周邊硬體

  • 提供 API 讓您更新軟體和韌體，還有能執行線上韌體更新的可能性

  • 綜合各種通訊、多媒體與使用者介面，支援：

  • GSM, GPRS and 藍牙 2.1 和 4.0 (由 MT2502A 支持)
  • 無線網路與衛星定位 (由配套晶片組支持)
  • MP3/ AAC 音訊播放
  • JPEG 解碼, 向量字型 (由 Etrump 提供)，以及其他多媒體功能

HDK

LinkIt Assist 2502

這塊與 Seeed Studio 合作設計的開源硬體開發板，包含了LinkIt Assist 2502 模組和多功能的 Wi-Fi與 GNSS模組。它還提供了可替換的觸控顯示螢幕，以及可連線外部硬體的完整接腳輸出入介面。


只需要一塊板子，就可得到全面的通訊和定位功能。

功能特點

  • 支援GSM, GPRS, Wi-Fi, 藍牙（ 2.1 SPP和 4.0 GATT）, GNSS (GPS, GLONASS 和 北斗) 和音訊輸出

  • 輸出接腳，包含數位I/O、類比I/O、PWM、I2C、SPI、UART與電源供應

  • 可替換的顯示器（包含一正方形LCM CTP）

  • 開源開發板硬體參考設計，包含原理圖、佈局圖與接腳細節

模組

LinkIt Assist 2502 模組

由 AcSiP 設計的三種模組：


  • LinkIt Assist 2502 模組 (內含聯發科技 MT2502，模組產品編號AcSiP AI2502S05)

這些模組能夠整合至您最終產品的PCB設計。

晶片

MT2502A

LinkIt Assist 2502 開發版的核心為聯發科技 MT2502A。


聯發科技 MT2502A 是一個功能豐富而強大的系統單晶片，具備高階 GSM/GPRS(2G) 的通訊能力，並且還整合了完全相容於 Blueooth v4.0 標準的 Bluetooth 硬體。


MT2502 還可以搭配聯發科技的 Wi-Fi(MT5931) 與 衛星定位(MT3332) 晶片組，為穿戴式與物聯網裝置提供高效能、低功耗的解決方案。

瞭解更多

SDK

LinkIt Assist 2502 SDK 2.0 提供一個 Eclipse IDE (with CDT) 的外掛，讓您進行軟體開發。該軟體開發包包含了韌體更新工具和 LinkIt Assist 2502 開發板軟體上傳工具。

功能特點

  • C 語言 API 開發環境

  • 編譯專案至LinkIt Assist 2502 執行文件 (.vxp)

  • LinkIt Assist 2502 API 函式庫提供以下通訊功能

  • TCP 連線
  • HTTPS
  • Bluetooth 4.0 GATT 與其他功能

  • 支援向量字型檔（由驛創提供），JPEG解碼，影像，並顯示在螢幕上

  • 與 Eclipse IDE (Indigo) CDT 相容（8.0.2 版或更新）

  • 支援微軟Windows XP，Vista，7 和 8

注意

LinkIt Assist 2502 SDK 亦可用於LinkIt ONE 開發板，唯部分功能可能受到硬體限制。請參閱LinkIt Assist 2502 開發者指南瞭解詳情。
