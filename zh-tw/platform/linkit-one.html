概述

聯發科技LinkIt™ ONE 是一個穿戴式與物聯網（IoT）裝置開發平台，提供近似於 Arduino 的硬體和 API ， 讓您針對您的創意量身打造裝置原型。

功能特色

  • 全方位通訊功能：基於聯發科技 MT2502A系統單晶片，支持各種通訊與多媒體功能，支援 GSM、GPRS、藍牙 2.1 和 4.0 、SD 卡、MP3/AAC 音訊，並整合Wi-Fi 和衛星定位系統

  • 與 Seeed Studio 合作的 LinkIt ONE 開發板，擁有與 Arduino UNO 類似的引腳，讓您連接許多不同的周邊元件

  • LinkIt ONE SDK提供開發者熟悉的 Arduino 環境和簡單易用的工具，讓初學者能輕鬆進行開發

開發板

LinkIt™ ONE

硬體開發套件內含有 LinkIt ONE 開發板、Wi-Fi與藍牙天線，GSM (2G)天線、GPS/GLONASS天線，以及鋰電池，您能很快的上手，實現您的構想。


您能替裝置加上額外的 Grove 感應器或者任何與開發板相容的周邊配件。


與 Seeed Studio 合作設計，此開發套件有您入門所需要的一切。

Key features

  • 支援 GSM, GPRS, Wi-Fi, 藍牙（包含2.1 SPP 和 4.0 GATT 協議），衛星定位 (GPS,GLONASS 和北斗)、音訊輸出，和SD 卡介面

  • 與 Arduino UNO 相似的接腳排列，包含數位與類比輸出入、PWM、I2C、SPI、UART 與電源供應

  • 開源硬體參考設計，提供原理圖，佈局和接腳細節

AWS IoT Starter Kit

此工具套件提供 LinkIt ONE 開發板，以及由 Seeed Studio 提供的電池和天線，還有多種 Grove 感應器。


搭配 LinkIt ONE SDK，以及mbed TLS 程式庫 和 AWS IoT軟體開發套件，您可以快速地實現穿戴式與物聯網概念，並利用 AWS 雲服務建立物聯網裝置。

晶片

MT2502A

LinkIt ONE 開發版的核心為聯發科技 MT2502A。


聯發科技 MT2502A 是一個功能豐富而強大的系統單晶片，具備高階 GSM/GPRS(2G) 的通訊能力，並且還整合了完全相容於 Blueooth v4.0 標準的 Bluetooth 硬體。


MT2502 還可以搭配聯發科技的 Wi-Fi(MT5931) 與 衛星定位(MT3332) 晶片組，為穿戴式與物聯網裝置提供高效能、低功耗的解決方案。

瞭解更多

SDK

LinkIt ONE SDK (for Arduino) 是 Arduino IDE 的擴充套件，此套件提供可靠且有彈性的方法來協助您在 LinkIt ONE 開發板上面構思與驗證您的穿戴式與物聯網裝置構想。

功能特色

  • 編譯 Arduino Sketches

  • 支援 Microsoft Windows XP, Vista, 7, 8 and 10，以及 Apple Mac 10.09 and 10.10

  • 提供 LinkIt ONE API，支援核心的 Arduino 功能，以及 LinkIt ONE 專有的通訊功能

  • 通話與 GPRS 數據傳送 API

  • 衛星定位 (GPS, GLONASS and BeiDou) API

  • Bluetooth SPP API

  • BLE GATT API

  • Wi-Fi API，並提供 TCP and UDP 連線功能
