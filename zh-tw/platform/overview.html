開發平台

LinkIt™ 平台

LinkIt 7697

具備 Wi-Fi 與低功耗藍牙連線能力，可使用 Arduino IDE 或基於 FreeRTOS 的 LinkIt SDK 開發。

LinkIt Smart 7688

打造有無線網路功能的智慧家庭與辦公室裝置原型。運行 OpenWrt Linux，可支援 Python, Node.js 與 C。

LinkIt ONE

用 Arduino IDE 在這張具備多樣連線功能的開發版實驗您的創意構想。

舊有開發平台

LinkIt Connect 7681

已停止銷售 - 內容維護至 2018 八月

打造具備 Wi-Fi 功能，可連線至智慧型手機與雲端服務的智能插座、燈泡等智能家電。

LinkIt Assist 2502

已停止銷售 - 內容維護至 2018 八月

製作精巧而具備可穿戴尺寸規格的概念雛形。提供 C 語言開發者功能豐富的 Eclipse 開發環境。

Helio™ 平台

MediaTek X20 Development Board

具備十核心 Helio X20 處理器的先進 96Boards 開發板。

雲端服務

MediaTek Cloud Sandbox

MediaTek Cloud Sandbox (MCS) 是一個雲端資料服務平台，讓您能夠連結物聯網裝置。
