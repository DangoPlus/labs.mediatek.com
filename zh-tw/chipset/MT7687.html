MT7687

入門指南
開發資源 API 指南
討論區 FAQ

概述

聯發科技 MT7687F 是高度整合的系統單晶片，包含一個微控制器，低功耗 1T1R 802.11n 2.4GHz Wi-Fi 子系統，以及 PMU。請查看比較表格以了解不同智慧家庭晶片間的差異。

主要功能

  • 具備 4Kbit eFuse，可存放裝置特定資訊和射頻校準資料

  • 高度整合的射頻 + 微控制器 + 記憶體

  • 低功耗的 RTC 模式

MT7687 Product Brief
MT7687 Datasheet
MT7687 Reference Manual

微控制器為含浮點運算功能的 ARM Cortex-M4，帶有內置的 SRAM/ROM 和串列快閃記憶體。它支援許多週邊介面，包括 UART、I2C、SPI、I2S、PWM、IrDA 和 ADC。


Wi-Fi 子系統支持 1T1R 802.11b/g/n 無線協定、基帶和 MAC，含有一個可以完全卸載微控制器運算負擔的網路處理器，旨在滿足低功耗和高輸送量的應用程式。

功能

開發平台

  • 含浮點運算功能的 ARM Cortex-M4 MCU

  • 內置 352KB SRAM 和 64KB boot ROM

  • 支援外掛帶有四週邊介面（QPI）模式的串列快閃記憶體

  • 在 XIP 模式下的 32KB cache

  • 支援在快閃記憶體上直接執行程序 （XIP）

  • 25 個 DMA channel

  • 支援 32kHz 晶體的低功耗 RTC 模式

安全啟動與加密引擎

  • 從串列快閃記憶體安全啟動

  • 硬體加密引擎包含 AES，DES/3DES，網路安全的 SHA2

連通性

  • SPI, UART, I2C, I2S, PWM, ADC, IrDA 和 GPIO

無線網路（Wireless Local Area Network）

  • 相容 IEEE 802.11 b/g/n

  • 專用的高性能 32-bit RISC CPU ，可卸載微控制器的運算負擔

  • 在 2.4 GHz 頻帶支援 20 MHz 與 40MHz 頻寬

  • 單頻 1T1R 模式，資料速率高達 150Mbps

  • 支援 STBC 與 LDPC

  • 支援 Greenfield 模式，混合模式與傳統模式

  • 支援 IEEE 802.11 e

  • 基於 WFA WPA/WPA2 personal，WPS2.0 的安全支援

  • 支援 802.11w 保護託管的幀

  • 支援 WFA WMM 的服務品質（QoS）管理

  • 集成低雜訊放大器（Low Noise Amplifier, LNA）、 功率放大器和 T/R 開關

  • 可選的對外掛的低雜訊放大器（Low Noise Amplifier, LNA）和功率放大器的支援

  • 支援 RX 多樣性以及額外 RX 輸入

技術和封裝

  • 8 x 8 毫米，68針 QFN 封裝，SiP

HDK

LinkIt 7687

LinkIt 7687 HDK（由品佳集團提供）是一塊針對物聯網應用的開發板，可用於設計與評估物聯網裝置。它基於MT7687F 系統單晶片，在 QFN68 的封裝中，具有含浮點運算的 ARM Cortex-M4 微控制器，並整合了802.11b/g/n Wi-Fi 無線網路功能。開發板提供豐富的周邊連接能力，能在連接至網路雲端服務的情況下，即時控制各種周邊硬體。

AI7687H 是一個高度整合的 SiP 模組，採用 MT7687，且模組大小僅有 18x18mm。

PT7687W-S 模組採用郵票貼方式設計，提供內建天線和 RF Connector 兩種天線樣式供選擇， 並將尺寸大小最佳化，尺寸大小為18.5mm x 32.5mm x 3.1mm。除了硬體設計外，此模組也提供內建雲商平台，協助產品快速開發服務。
