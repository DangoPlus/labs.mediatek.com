MT7688

概述

聯發科技 MT7688AN 系統單晶片可應用於家庭自動化的橋接中心。它整合了 1T1R 802.11n Wi-Fi radio、580MHz MIPS® 24KEc™ CPU、1-port fast Ethernet PHY、USB2.0 host、PCIe、SD-XC、 I2S/PCM，並支援多種低速輸出入介面在單一顆系統單晶片當中。請查看比較表格以了解不同智慧家庭晶片間的差異。


MT7688 支援兩種運作模式：IoT gateway 模式與 IoT device 模式.


在 IoT gateway 模式中，可透過 PCIe 界面連接至 802.11ac 晶片組，並作為雙頻 802.11ac 同步閘道。高速的 USB 2.0 介面可讓 MT7688 連接至額外的 3G/LTE modem 硬體，或連接到 H.264 ISP 作為無線 IP 相機的應用。 IoT gateway 模式也支持觸控板、Bluetooth Low Energy、Zigbee/Z-Wave 和 Sub-1 GHz RF 等智慧家庭應用所需的硬體。


在 IoT device 模式下，MT7688 支援 eMMC、SD-XC 與 USB 2.0，並且可透過 192Kbps/24bits I2S 介面播放高音質的 Wi-Fi 音訊，或是透過 PCM 進行 VoIP 應用。同時也支援各種周邊介面，包含 PWM、 SPI slave、三組 UART 與更多 GPIO 介面。

功能特點

  • MIPS24KEc 580 MHz，具備 64 KB I-Cache 與 32 KB D-Cache

  • 支援最大達 256MB RAM ，並支援 SD-card 儲存媒介

  • 1T1R 2.4 GHz，具備 150 Mbps PHY data rate

  • Legacy 802.11b/g and HT 802.11n modes

  • 1-port 10/100 FE PHY

  • SD-XC, eMMC, I2C, PCM, I2S(192K/24bits), PWM, SPI master/slave, UART lite, JTAG 與 GPIO
