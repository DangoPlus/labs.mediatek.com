MT2601

概述

聯發科技 MT2601 為穿戴式裝置帶來雙核心效能，又不會對電池使用時間帶來負擔。MT2601 具備 1.2GHz 的雙核心 ARM Cortex-A7 CPU、ARM Mali-400 MP GPU、ePOP memory link，還有可在 qHD 解析度下帶來順暢表現的顯示控制器。


聯發科技 MT2601 能夠外接各種外部感測器，並可外掛 MT6630 無線通訊晶片，只需很小的 PCBA 尺寸，就能具備 Wi-Fi、衛星定位、藍牙與 ANT+ 功能。另外還相容於 MT6323 電源管理晶片以及 MT6605 NFC 晶片，帶來更多硬體配置彈性。


MT2601 的定位追蹤功能可滿足各種穿戴式裝置的需求，比如基於 Android Wear 的運動與健身裝置。針對娛樂影音方面的需求，它的多媒體系統具備了五百萬畫素的相機支援，以及 720p 的影片錄製與播放。


聯發科技 MT2601 具備領先群倫的能源應用效率，其高度整合的特質，能有效節省佈局大小與器件數量，為產品帶來更有效益的成本優勢。

功能特色

高效能與低功耗

  • 專為穿戴式裝置而設計，MT2601 具有一雙核 1.2GHz ARM Cortex-A7 處理器，在功耗與效能間取得最佳的平衡。此處理器與 Bluetooth Low Enegery 科技搭配，可進一步節省電池消耗。聯發科技 MT2601 即使在多工作業時，也能為使用者帶來驚喜。

高度整合解決方案

  • MT2601 不僅僅是一個處理器，它是一個高度整合的系統單晶片，具備多種通訊功能，還能額外擴充衛星定位功能（MT6630）與電源管理功能（MT6323），為所有小型通訊裝置提供了完整的多合一解決方案。

優異的多媒體表現

  • 透過它領先市場的多媒體子系統，聯發科技 MT2601提供完整的顯示系統，帶來舒適的娛樂體驗

為生態系做好準備

  • 除了可以編解碼 H.264 影片之外，聯發科技 MT2601 還很適合執行具備豐富影像和動畫內容的先進 Android Wear 應用程式。透過優異的軟硬體整合，MT2601 可帶來完整的 Android Wear 體驗。

產品規格

  • CPU: ARM Cortex-A7 1.2GHz

  • GPU: ARM Mali-400

  • 記憶體: eMMC + LPDDR2/3，最大可達 4GB

  • 通訊功能: Wi-Fi 802.11abgn/ac1T1R, BT 3.0+HS, BLE 4.1, ANT+, GPS / Glonass / Beidou / Galileo / QZSS (需搭配 MT6630)

  • 相機: 支援達 5MP 主相機與 1.3MP 副相機

  • 顯示: 最大支援至 qHD（解析度 960x540）

  • 整合性: 相容於 3G modem, MT6605 NFC 與 MT6323 電源管理晶片。周邊介面支援感測器，記憶卡與外部的 sensor hub。
