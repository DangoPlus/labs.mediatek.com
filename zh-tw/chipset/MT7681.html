MT7681

概述

聯發科技 MT7681 是個高整合的無線網路系統級晶片（SoC）並擁有物聯網裝置所需的嵌入式TCP/ IP協定堆疊。IoT 裝置經由 MT7681 將可以連線到其他智慧裝置或雲端應用和服務。請查看比較表格以了解不同智慧家庭晶片間的差異。


在 Wi-Fi Station 模式時，MT7681先連線到無線 AP，之後可將與網站服務或雲端伺服器進行通訊。一個典型的用途是，使用者可透過一個智慧家庭的網站來控制他們的家的暖氣機。

功能特點

  • 支援 Wi-Fi Station 模式: 802.11b/g/n

  • 支援Wi-Fi Access Point 模式: 802.11b/g

  • Smart Connection API 讓您容易的透過安卓或iOS應用程式來設定無線網路

  • TCP/IP 協定

  • 透過UART進行韌體升級, 並提供空中韌體升級 (FOTA) API

  • 軟體PWM

LinkIt Connect 7681

此 LinkIt Connect 7681 開發板是與 Seeed Studio 協同設計的.它提供您五個 GPIO 接腳和一個UART 埠 – 並各有專屬的 LED – 讓您快速連線周邊硬體和外部電路。本開發板基於聯發科技 MT7681，是具備高整合度的無線 SoC，並帶有 TCP/ IP 協定堆疊。此開發板上含有和 AcSiP 協同設計的 LinkIt Connect 7681 模組，這個模組只有15×18 毫米，也可分別由 Seeed Studio 和AcSiP 批發商獲得，並直接整合至您的最終產品的 PCB 上。
