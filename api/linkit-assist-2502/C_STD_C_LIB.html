Description

Provides functions of string operations, such as string conversion and comparison.

Introduction

There are two type of strings are allowed in this platform. They are byte-strings and word-strings. A byte-string is a string that each character is one byte in size. A word-string is a string that each character is one word (two bytes) in size. 

The functions provided in the module allow the application to convert a string to integer, upper or lower case, to compare two strings, to search for a string pattern, to copy a string, to concatenate two strings, etc.
