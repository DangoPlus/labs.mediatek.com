);
Description

This function creates UDP socket.

Parameters
Parameters 
Description 
VMINT port 
[IN] The port number for binding. 
VM_BEARER_DATA_ACCOUNT_TYPE data_account_type 
[IN] Is a Type that defines how to connect, please see VM_BEARER_DATA_ACCOUNT_TYPE for details. 
vm_udp_callback callback 
[IN] The Callback function for UDP notifications. 
VMBOOL loopback 
[IN] The loopback setting, VM_TRUE: loopback; VM_FALSE: no loopback. 
Returns

The create UDP is successful or not.

Return Values
Return Values 
Description 
>=0 
The UDP handle.
 
< 0 
An error code in VM_UDP_RESULT.
