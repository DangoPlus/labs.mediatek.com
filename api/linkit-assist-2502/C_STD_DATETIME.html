Description

Sets and retrieves system time, or gets the microsecond counts since the system is powered on.

Introduction

The functions in the module can be used to set system time or get system time, or get the counts from microsecond timer since the system is powered on or calculate out the duration of between the microsecond counts.

Limitation

Do not use the simple subtraction to calculate the duration of microsecond counts. Use the vm_time_ust_get_duration() to get the duration instead, because the counts of the microsecond timer may wrap around.

vm_time_ust_get_count 
Gets the count from the microsecond timer (ust), a hardware timer. This microsecond timer counts the microseconds passed from the time the system is powered on until now. It allows the caller to measure the durations in high precision. To calculate the duration between different timer counts, use vm_time_ust_get_duration(), instead of simply subtract values, because the timer counts may wrap around. 
vm_time_ust_get_duration 
Gets the duration from start_count to end_count, in microsecond. The caller should ensure that the end_count is equal or greater than the start_count.
