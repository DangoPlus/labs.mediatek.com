Description

Provides functions to access the information from the SIM card in the device.

Introduction

With the functions provided in the module, an application can check whether there is a SIM card in the device, whether the SIM card is active, what the status of the SIM card is. It can also get the IMEI, IMSI, ICC ID, PLMN of the SIM card.

Limitation

The LinkIt ONE v2.0 SDK supports only one SIM card. The functions dealing with multiple SIM cards are not applicable for the LinkIt ONE HDK/SDK.
