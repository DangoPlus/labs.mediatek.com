Power Management

Description

Allows shutting down and rebooting the system, also provides battery and charging information.

Introduction

The power module provides basic APIs for system shutdown and reboot. There are also APIs for querying the current battery charging status and current battery level.

Functions
Function 
Description 
vm_pwr_get_battery_level 
Get a calculated battery level. This battery level is calculated based on the battery voltage and then feed to an algorithm to ensure the level is monotonically decreasing. 
vm_pwr_is_charging 
Get the current battery charging status. 
vm_pwr_lcd_sleep_in 
LCD module enter sleep mode. 
vm_pwr_lcd_sleep_out 
LCD module leave sleep mode. 
vm_pwr_reboot 
Reboot the system. The system will restart after normal shutdown flow, there is no need to press the power key. The applcation does not receive VM_EVENT_QUIT event. Therefore developers should ensure vital information are stored properly before calling this function. 
vm_pwr_scheduled_startup 
this interface will set device auto power on option, include time and check rules. the option will not save always, you need set it again after device reboot. 
vm_pwr_shutdown 
Shutdown the system. The shutdown flow will uninitialize the LinkIt runtime directly. The applcation does not receive VM_EVENT_QUIT event. Therefore developers should ensure vital information are stored properly before calling this function.
