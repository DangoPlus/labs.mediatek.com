Description

Queries information such as version and application name from LinkIt applications.

Introduction

The Tag module allows developers to embed certain informations into their LinkIt applications, including application name, version, and developer information. These information embedded at the end of the vxp executables are called tags. A tag can be of integral type or of string type. To define tag information, use the project setting in the Eclipse plug-in. To retrieve the tag information in LinkIt application, call function vm_tag_get_tag with VM_TAG_ID as parameter.

Limitation

Some tag IDs may not be available on applications generated by different versions of LinkIt SDK.
