MT2523

入门指南
技术资源 API 指南
讨论区 FAQ

概述

联发科技 MT2523 产品系列（MT2523D/MT2523G）具备高度集成系统封装（SiP）， 包含一个微控制器单元，低功耗全球导航卫星系统（MT2523G），双模蓝牙以及一个电源管理单元（PMU）。微控制器单元是一个 ARM Cortex-M4F MCU，内置 4 MB PSRAM 和 4 MB 闪存。MT2523 还支持 UART、 I2C、 SPI、 I2S、 PWM、 SDIO、 MSDC、 USB、 PCMIF、 ADC 和双数字麦克风等接口。

主要功能

  • 射频 + 单片机 + 内存的高集成

  • 低功耗 RTC 模式

  • 支持多种频率和电压模式，以实现改进和延长电池寿命，同时保持高性能

  • 可在 10μs 内从睡眠模式快速切换到活动模式

MT2523 Product Brief
MT2523 Datasheet
MT2523 Reference Manual

全球导航卫星系统的辅助芯片是多-全球导航卫星系统射频 SoC，以最低的功率消耗达到行业的最高水平的灵敏度、 准确度和第一次定位时间（TTFF）。它支持不同的定位和导航应用程序，包括自主的 GPS、GLONASS、伽利略、北斗、SBA 测距（WAAS、EGNOS、GAGAN 和 MSAS）、QZSS、DGPS（RTCM）和辅助全球定位系统（A-GPS）。它还包含高达 12 多音主动干扰消除器。


双模式蓝牙子系统，支持蓝牙 2.1 和低功耗蓝牙 4.2，包含蓝牙无线电、基带、链接控制器。


显示组件支持 MIPI-DSI 和串行接口，它包含这些 2D 功能：真正的颜色显示，每个像素的 alpha 通道和抗锯齿的字体，以及 1 位索引颜色以节省内存和运算能力。相机组件支持 DDR 模式，最大时钟速率 48 MHz，分辨率最大为 VGA。

功能

开发平台

  • 基于 FPU 的 ARM Cortex M4F MCU

  • 内置 4 MB + 160 KB SRAM 和 4 MB 的闪存

  • 25 个 DMA channel

  • 支持 32kHz 晶体的低功耗 RTC 模式

  • 双模蓝牙BT/BLE，大 TX 功率和优良的 RX 灵敏度，以支持最佳覆盖

  • 支持高电池电压 （4.8 v），延长电池寿命

  • 多种频率和电压模式以支持低功耗和计算功率平衡：高速（1.3 v,208 MHz），全速（1.1v,104 MHz），低速（0.9 v,26 MHz）

  • 低至 40μA 的低功耗晶体振荡器时钟源

连通性

  • UART, I2C, SPI, I2S, PWM, SDIO, MSDC, USB, PCMIF, ADC，双数字麦克风

全球导航卫星系统（仅限2523G）

  • 支持多个卫星系统，包含GPS/GLONASS/伽利略/北斗

  • 支持多-全球导航卫星系统包括卫星系统（Quasi Zenith Satellite System），星基增强系统（Satellite Based Augmentation System）测距

  • 支持 WAAS/EGNOS/ MSAS/GAGAN

  • 获得 ISSCC 2011 奖项的 12 频道主动多音干扰消除器

  • RTCM

  • 室内和室外的多路径检测和补偿

  • 支持 FCC E911 法规和 A-GPS

  • 最大固定更新率达 10 Hz

蓝牙

  • 蓝牙 2.1 + EDR

  • 蓝牙 4.2 低功耗（LE）

  • 集成 BALUN 和 PA

  • 分散网支持：最多支持 7个微微网同时做后台的查询/扫描

  • 最多同时支持 7 个 ACL link

  • 支持 SCO link 和带有重传功能的 eSCO link

  • 信道质量驱动的数据速率自适应

  • AFH 的信道评估

显示

  • DBI 接口：同时连接到 2 个串行液晶显示模块，最大分辨率可达 320 x 320

  • DSI 接口：支持同步事件，同步脉冲，和突发模式视频模式，并且支持超低功耗的低帧速率（LFR）模式，和高达 500 Mbps 的高速模式

技术和封装

  • 高度集成55纳米超低功耗技术

  • MT2523D 是6.2 毫米 x 5.8 毫米 165-ball VFBGA 包，包含 SiP 串行闪存和 PSRAM

  • MT2523G 是9.2 毫米 x 6 毫米 242-ball VFBGA 包，包含 SiP 串行闪存、PSRAM 和全球导航卫星系统

HDK

LinkIt 2523

LinkIt 2523 HDK 是物联网和穿戴的全功能的演示板，采用联发科技 MT2523G，基于ARM Cortex-M4内核的微控制器单元（MCU）。它支持丰富的连接功能，包含与云服务的通信以及实时控制。LinkIt 2523 HDK 支持在 Keil IDE上的开发，并提供额外的库来集成各种外围设备，并支持一类新的高度连接的应用程序。
