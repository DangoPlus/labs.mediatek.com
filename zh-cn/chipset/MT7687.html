MT7687

入门指南
技术资源 API 指南
讨论区 FAQ

概述

联发科技MT7687F 是高度集成的单芯片，包含一个应用处理器，低功耗 1 x 1 802.11n 单频 Wi-Fi 子系统，电源管理单元（PMU）。请查看比较表格以了解不同智能家庭芯片间的差异。

主要功能

  • 具集成 4Kbit eFuse 来存储设备特定信息和射频校准数据

  • 射频 + 单片机 + 内存的高集成

  • RTC 的低功耗模式

MT7687 Product Brief
MT7687 Datasheet
MT7687 Reference Manual

应用程序处理器子系统包含 ARM Cortex-M4F MCU。它还包括许多外围设备，包括 UART、I2C、SPI、I2S、PWM、IrDA 和辅助模数转换器。它还包括内置的 SRAM/ROM 和串行闪存。


Wi-Fi 子系统包含 802.11b/g/n 无线协议、基带和 MAC，旨在满足低功耗和高吞吐量的应用程序。它还包含一个可以完全卸载应用程序处理器的网络处理器。

功能

开发平台

  • 基于 FPU 的 ARM Cortex M4F MCU

  • 內置 352KB SRAM 和 64KB boot ROM

  • 支持外挂带有四外围接口 (QPI) 模式的串行闪存

  • 支持在 flash 闪存上的芯片内执行（XIP）

  • 在芯片内执行（XIP）模式下的 32KB cache

  • 25 个 DMA channel

  • 支持 32kHz 晶体的低功耗 RTC 模式

安全启动与加密引擎

  • 从串行闪存安全启动

  • 硬件加密引擎包含 AES，DES/3DES，网络安全的 SHA2

连通性

  • SPI, UART, I2C, I2S, PWM, ADC, IrDA, GPIO

无线局域网（Wireless Local Area Network）

  • 相容 IEEE 802.11 b/g/n

  • 专用的高性能 32-bit RISC CPU ，可卸载微控制器的运算负担

  • 在 2.4 GHz 频带支持 20 MHz 和 40MHz 频宽

  • 单频 1T1R 模式，数据速率高达 150Mbps

  • 支持 STBC 和 LDPC

  • 支持 Greenfield 模式，混合模式，传统模式

  • 支持 IEEE 802.11e

  • 基于 WFA WPA/WPA2 personal 和 WPS2.0 的安全支持

  • 支持 802.11w 保护托管的帧

  • 支持 WFA WMM 的服务品质（QoS）管理

  • 集成低噪声放大器（Low Noise Amplifier, LNA）、 功率放大器和 T/R 开关

  • 可选的对外挂的低噪声放大器（Low Noise Amplifier, LNA）和功率放大器的支持

  • 支持 RX 多样性以及额外 RX 输入

技术和封装

  • 8 毫米 × 8 毫米 68-pin QFN 封装与 SiP

HDK

LinkIt 7687

LinkIt 7687 HDK 是一种低成本和易于使用物联网（IoT）的开发平台，使用实时操作系统（RTOS）来设计、试制、评估和实施物联网（IoT）项目。它是采用MT7687F，基于ARM Cortex-M4F MCU 的 QFN68 封装。HDK 拥有丰富的连接功能，包含与云服务的通信以及实时控制。LinkIt 7687 HDK 提供额外的库来集成各种外围设备，并支持一类新的高度连接的应用程序。

模块

AI7687H （由 AcSiP 提供）

AI7687H 是一个高度集成的 SiP 模块，采用 MT7687，且模块大小仅有 18x18mm。

MT7687MODULE 模块 （由 SAC 提供）

此模块是基于联发科技的 MT7687 模块参考设计而制作，并且使用于 LinkIt 7687 HDK 之上。

更多细节

PT7687W-S（由 PTCOM 提供）

PT7687W-S 模块采用邮票贴方式设计，提供内置天线和 RF Connector 两种天线样式供选择， 并将尺寸大小最佳化，尺寸大小为18.5mm x 32.5mm x 3.1mm。除了硬件设计外，此模块也提供内置云商平台，协助产品快速开发服务。
