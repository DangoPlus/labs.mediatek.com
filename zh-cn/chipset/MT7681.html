MT7681

概述

联发科技 MT7681 是个高集成的无线网络系统级芯片（SoC）并拥有物联网设备所需的嵌入式TCP/ IP协定堆叠。IoT 设备经由 MT7681 将可以连接到其他智能设备或云端应用和服务。请查看比较表格以了解不同智能家庭芯片间的差异。


在 Wi-Fi Station 模式时，MT7681先连接到无线 AP，之后可将与网站服务或云端服务器进行通信．一个典型的用途是，用户可透过一个智能家居的网站来控制他们的家的暖气机。

功能特点

  • 支持 Wi-Fi Station 模式: 802.11b/g/n

  • 支持Wi-Fi Access Point 模式: 802.11b/g

  • Smart Connection API 让您容易的透过安卓或iOS应用程序来设置无线网络

  • TCP/IP 协定

  • 透过UART进行固件升级, 并提供空中固件升级 (FOTA) API

  • 软件PWM

LinkIt Connect 7681

此 LinkIt Connect 7681 开发板是与 Seeed Studio 协同设计的。它提供您五个 GPIO 接脚和一个UART 端口 – 并各有专属的 LED – 让您快速连接周边硬件和外部电路。本开发板基于联发科技 MT7681，是具备高集成度的无线 SoC，并带有 TCP/ IP 协定堆叠。此开发板上含有和 AcSiP 协同设计的 LinkIt Connect 7681 模块，这个模块只有15×18 毫米，也可分别由 Seeed Studio 和AcSiP 批发商获得，并直接集成至您的最终产品的 PCB 上。
