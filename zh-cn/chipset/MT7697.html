MT7697

入门指南
技术资源 API 指南
讨论区 FAQ

概述

MT7697 是高度集成的系统单芯片，包含有一个 Cortex-M4 单片机，低功耗 1T1R 802.11b/g/n 2.4GHz Wi-Fi 子系统，低功耗蓝牙子系统，以及 PMU。MT7697D 是强化版本，额外提供了 802.11a，以及 5GHz 频带的支持。请查看比较表格以了解不同智能家庭芯片间的差异。

MT7697 主要功能

  • 具备 4Kbit eFuse，可存放装置特定资讯和射频校准资料

  • 先进的 Wi-Fi 与蓝牙共存方案

  • 高度整合的射频 + 微控制器 + 内存

  • 低功耗的 RTC 模式

MT7697 Product Brief
MT7697 Datasheet

应用程序处理器子系统包含 ARM Cortex-M4F MCU。它还包括许多外围设备，包括 UART、I2C、SPI、I2S、PWM、IrDA 和辅助模数转换器。它还包括内置的 SRAM/ROM。


Wi-Fi 子系统包含 802.11b/g/n 无线协议、基带和 MAC，旨在满足低功耗和高吞吐量的应用程序。它还包含一个可以卸载应用程序处理器的网络处理器。

MT7697D 主要功能

MT7697 和 MT7697D 之间的区别在于，MT7697D 支持下面的額外功能：

  • IEEE 802.11a

  • 使用 5 GHz 频段

  • 双频 1T1R

MT7697D Product Brief
MT7697D Datasheet

功能

开发平台

  • 带有 FPU 的 ARM Cortex-M4 MCU，最大运行时脉为 192 MHz

  • 內置 352KB SRAM 和 64KB boot ROM

  • 支持外挂带有四外围接口 (QPI) 模式的串行闪存

  • 支持在 flash 闪存上的芯片内执行（XIP）

  • 在芯片内执行（XIP）模式下的 32KB cache

  • 25 个 DMA channel

  • 支持 32kHz 晶体的低功耗 RTC 模式

安全启动与加密引擎

  • 从串行闪存安全启动

  • 硬件加密引擎包含 AES，DES/3DES，网络安全的 SHA2

连通性

  • SPI, UART, I2C, I2S, PWM, ADC, IrDA, GPIO

技术和封装

  • 高度集成的 40nm RFCMOS 技术

  • 8 × 8 毫米 68-pin QFN 封裝

无线局域网（Wireless Local Area Network）

  • 相容 IEEE 802.11 b/g/n

  • 专用的高性能 32-bit RISC CPU

  • 在 2.4 GHz 频带支持 20 MHz 与 40MHz 频宽

  • 单频 1T1R 模式，数据速率高达 150Mbps

  • 支持 STBC 与 LDPC

  • 支持 Greenfield 模式，混合模式与传统模式

  • 支持 IEEE 802.11 d/e/h/i/k/r/w

  • 基于 WFA WPA/WPA2 personal，WPS2.0 与 WAPI 的安全支持

  • 支持 802.11w 保护托管的帧

  • 支持 WFA WMM，WMM PS 的服务品质（QoS）管理

  • 集成低噪声放大器（Low Noise Amplifier, LNA）、 功率放大器和 T/R 开关

  • 可选的对外挂的低噪声放大器（Low Noise Amplifier, LNA）和功率放大器的支持

  • 支持 RX 多样性以及额外 RX 输入

藍牙

  • 蓝牙 4.2 低能量（LE）

  • 集成 BALUN 和 PA

  • 支持 SCO link 和帶有重傳功能的 eSCO link

  • AFH 的信道评估

HDK

LinkIt 7697

LinkIt 7697 是一块针对物联网应用的开发板。它基于 MT7697 系统单芯片，具有含浮点运算的 ARM Cortex-M4 单片机，并集成了 802.11b/g/n Wi-Fi 无线网络与 Bluetooth 4.2 低功耗蓝牙。

功能特色

  • 基于ARM Cortex-M4 的 MT7697 系统单芯片，时脉 192MHz

  • 802.11b/g/n 无线网络支持

  • Bluetooth 4.2 低功耗蓝牙

  • 352 KB RAM

  • 4 MB 外置闪存

  • Micro-USB 接头用于供电

  • 透过版载的 CP2102N USB 转 UART 芯片提供 COM 串行口，用于刻录

  • 提供系统重置按钮与用户自定义按钮

  • 提供下列周边接口：GPIO、UART、I2C、SPI、PWM、EINT、ADC、IrDA 与 I2S

  • 提供 SWD 调试接口排针

  • 使用 LinkIt SDK 开发，可支持 GCC、IAR Embedded Workbench 以及 Keil uVision IDE (使用指南)

  • 提供额外扩充包支持 Arduino IDE

  • 尺寸：48 x 26 mm

模块

AI7697H 与 AI7697HD （由 AcSiP 提供）

AI7697H 是一个高度集成的 SiP 模块，采用 MT7697，并且在模块上内置了 4MB 闪存，以及 MHF4 接头，且模块大小仅有 18x18mm。AI7697HD 则是采用 MT7697D 的版本。

wrtnode^7 （由 wrtnode 提供）

wrtnode^7 是全功能的 MT7697 模块，含有 4MB flash，提供内置陶瓷天线，也可透过 IPEX MHF4 外置天线，并在 23*18mm 的小尺寸中，提供 MT7697 的所有接脚。

了解更多

PT7697WB-S 与 PT7697WB-D（由 PTCOM 提供）

PT7697WB-S 模块采用邮票贴方式设计，提供内置天线和 RF Connector 两种天线样式供选择， 并将尺寸大小最佳化，尺寸大小为18.5mm x 32.5mm x 3.1mm。除了硬件设计外，此模块也提供内置云商平台，协助产品快速开发服务。
