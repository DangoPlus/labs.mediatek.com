LinkIt™ Smart 7688

概述

LinkIt Smart 7688 开发版基于联发科技的 MT7688AN 系统级芯片，提供高性能的 802.11n 1T1R Wi-Fi 和各种周边连接能力。软件开发环境基于 OpenWrt 操作系统，默认安装了许多物联网设备开发常用的套件，包含了多种编程语言的支持，例如 Python、Node.js 和 C 语言。 LinkIt Smart 7688 Duo 开发版则额外搭配了 ATmega32U4 单片机，可使用 Arduino 开发环境连接更多周边。

开发板

LinkIt Smart 7688 开发平提供硬件开发包 (HDKs) 让您实现物联网概念。该平台提供两个 HDK。

LinkIt Smart 7688

使用 MT7688AN 作为微处理器 (MPU) 。 支持 Python、Node.js 和 C 语言开发。

功能特点

  • 支持 Wi-Fi、USB host 和 SD 卡

  • 接脚提供的规格包含： PWM、I2C、SPI、I2S 和 UART (Lite)

  • 开源的硬件参考设计：包含原理图、布局图、和接脚细节

LinkIt Smart 7688 Duo

LinkIt Smart 7688 Duo开发板使用 MT7688AN 搭配 ATmega32U4 单片机 (MCU) 。


除了支持 Python、Node.js 和 C 语言之外，也支持以 Arduino 开发环境进行 MCU 端的应用程序开发。

功能特点

  • 支持 Wi-Fi, USB host 和 SD 卡

  • MT7688AN接脚：Ethernet、USB Host 和 UART

  • MCU 接脚: ADC、PWM、I2C、SPI 和 UART

  • 开源的硬件参考设计：包含原理图、布局图、和接脚细节
