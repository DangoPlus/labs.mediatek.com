LinkIt™ 开发平台 for RTOS

概述

联发科技 LinkIt™ 开发平台 for RTOS 是一个具备丰富功能的物联网与穿戴式开发平台。该平台支持一系列联发科技芯片，包含用于物联网设备的MT7687F, MT7697 and MT7697D，以及用于穿戴式设备的MT2523D 和 MT2523G。本开发平台基于受欢迎的FreeRTOS操作系统，透过提供统一的程序接口与开发工具，您将能在多样的系统单芯片上开发广泛的消费性或商务物联网设备。

连网系统单芯片

一系列多样化的系统单芯片（SoC），集成了无线通信功能与单片机，具有优秀功耗性能表现。


  • Wi-Fi 无线网络 (MT7687 family)
  • Wi-Fi 无线网络与Bluetooth LE低功耗蓝牙 (MT7697 family)
  • 卫星定位与双模式蓝牙 Bluetooth EDR/LE(MT2523 family)

广受欢迎的软硬件架构

基于流行的ARM Cortex-M4硬件架构以及FreeRTOS操作系统，您可集成并利用各种多样化的第三方工具，在LinkIt™ 开发平台 for RTOS上面进行开发。

丰富的软件功能

本平台提供LinkIt SDK v4，集成了多种中间件，可用于快速开发广泛类型的应用。软件开发套件提供的功能有：TCP/IP，SSL/TLS，MQTT，Bluetooth EDR/LE profiles，传感器子系统，电池管理，以及远程无线固件更新。

LinkIt 7697

LinkIt 7697 是一块针对物联网应用的开发板。它基于 MT7697 系统单芯片，具有含浮点运算的 ARM Cortex-M4 单片机，并集成了 802.11b/g/n Wi-Fi 无线网络与 Bluetooth 4.2 低功耗蓝牙。

功能特色

  • 基于ARM Cortex-M4 的 MT7697 系统单芯片，时脉 192MHz

  • 802.11b/g/n 无线网络支持

  • Bluetooth 4.2 低功耗蓝牙

  • 352 KB RAM

  • 4 MB 外置闪存

  • Micro-USB 接头用于供电

  • 透过版载的 CP2102N USB 转 UART 芯片提供 COM 串行口，用于刻录

  • 提供系统重置按钮与用户自定义按钮

  • 提供下列周边接口：GPIO、UART、I2C、SPI、PWM、EINT、ADC、IrDA 与 I2S

  • 提供 SWD 调试接口排针

  • 使用 LinkIt SDK 开发，可支持 GCC、IAR Embedded Workbench 以及 Keil uVision IDE (使用指南)

  • 提供额外扩充包支持 Arduino IDE

  • 尺寸：48 x 26 mm

LinkIt 7687 HDK by SAC

LinkIt 7687 HDK （由品佳集团提供）是一块针对物联网应用的开发板，可用于设计与评估物联网设备。它基于MT7687F系统单芯片，在QFN68的封装中，具有含浮点运算的 ARM Cortex-M4单片机，并集成了802.11b/g/n Wi-Fi 无线网络功能。开发板提供丰富的周边连接能力，能在连接至网络云端服务的情况下，即时控制各种周边硬件。

本开发板搭配LinkIt SDK v4，含有多样的网络连接功能，能协助您集成各种周边硬件至各种新类型的连网服务。

功能特色

  • 基于ARM Cortex-M4 的 MT7687F 系统单芯片

  • 802.11b/g/n 无线网络支持，具备内置天线

  • U.FL 接头可用于外置天线或进行测试

  • Micro-USB接头可用于供电，并提供串口与CMSIS-DAP接口，可进行开发与除错

  • 接受外部 1.8 ~ 3.2V 电源，用 两个AA电池就能驱动

  • 系统重置、RTC中断与外部中断，共三个按钮

  • 提供电流功耗量测脚位

  • 兼容于Arduino Uno (revision 3)的脚位配置，并提供额外八个扩充脚位

  • 尺寸： 6.05 x 10.85 公分

LinkIt 2523 HDK by SAC

LinkIt 2523 HDK （由品佳集团提供）提供您开发穿戴式设备原型所需的一切功能。它使用联发科技MT2523G，提供具备浮点运算能力的ARM Cortex-M4单片机，并集成Bluetooth EDR/LE，以及卫星定位系统。开发板上并提供触摸显示屏幕，以及为电池与传感器保留的扩充接头。

功能特色

  • 基于ARM Cortex-M4的MT2523G 系统单芯片

  • 支持Bluetooth EDR/LE与卫星定位系统

  • 内置天线，并可透过SMA 接头外置天线

  • Micro-USB接头可用于供电，并提供串口与CMSIS-DAP接口，可进行开发与除错

  • 具备320 x 320 像素的触摸显示屏

  • 喇叭与耳麦插孔

  • 4GB eMMC 闪存保存媒体

  • SD卡插槽

  • 电池、传感器与相机的扩充接脚

  • 电流功耗量测接脚

  • 尺寸： 7 x 21.5 公分

SDK

联发科技 LinkIt™ 软件开发包 v4 提供您开发应用所需要的软件与工具。本软件架构切分为三部分，由BSP（开发板支持软件包），中间件以及应用层所组成。


开发板支持软件包当中包含了系统单芯片的硬件驱动程序，以及 FreeRTOS 操作系统的集成。此层在不同的芯片之间提供了共通的硬件抽象层（HAL），并提供开发板上内附的周边硬件的驱动程序，如显示与触摸。


中间件层提供了应用程序开发所需要的重要功能模块，如Wi-Fi与Bluetooth的协议堆叠，IP实做（lwIP）与其他互联网协议，以及电池管理、传感器子系统，远程无线固件更新等功能。


应用层则提供了易于使用的范例程序，参考设计，以及示范专案，使开发者可以快速将之利用于本身的开发案中。


您可以选择使用免费的GCC工具炼开发，或使用专业的集成开发环境，如 Keil μVision 和 IAR Embedded Workbench 进行开发。


软件开发环境

  • ARM Keil μVision 扩充包 （适用于Windows）
  • IAR Embedded Workbench（适用于Windows）
  • GCC （适用于Linux，或安装MinGW的Windows）
