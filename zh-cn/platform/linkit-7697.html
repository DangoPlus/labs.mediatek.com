LinkIt™ 7697

概述

LinkIt 7697 是一块针对物联网应用的开发板。它基于 MT7697 系统单芯片，具有含浮点运算的 ARM Cortex-M4 单片机，并集成了 802.11b/g/n Wi-Fi 无线网络与 Bluetooth 4.2 低功耗蓝牙。根据您的需求，您可以选择不同的软件开发环境。


使用 Arduino IDE

支持基础的 Arduino API，并且提供额外的 Wi-Fi 与蓝牙函式库

使用 Blockyduino

使用 Blockly 可视化进程编辑工具，只要拖拉图形组件就可以简易的连接周边与云端服务

使用 LinkIt SDK

以 FreeRTOS 为核心，可使用 GCC、Keil uVision 或 IAR Embedded Workbench 开发

开发板

LinkIt 7697

LinkIt 7697 是一块针对物联网应用的开发板。它基于 MT7697 系统单芯片，具有含浮点运算的 ARM Cortex-M4 单片机，并集成了 802.11b/g/n Wi-Fi 无线网络与 Bluetooth 4.2 低功耗蓝牙。

功能特色

  • 基于ARM Cortex-M4 的 MT7697 系统单芯片，时脉 192MHz

  • 802.11b/g/n 无线网络支持

  • Bluetooth 4.2 低功耗蓝牙

  • 352 KB RAM

  • 4 MB 外置闪存

  • Micro-USB 接头用于供电

  • 透过版载的 CP2102N USB 转 UART 芯片提供 COM 串行口

  • 提供系统重置按钮与用户自定义按钮

  • 提供下列周边界面：GPIO、UART、I2C、SPI、PWM、EINT、ADC、IrDA 与 I2S

  • 提供 SWD 除错界面排针

  • 可使用 Arduino IDE

  • 尺寸：48 x 26 mm
