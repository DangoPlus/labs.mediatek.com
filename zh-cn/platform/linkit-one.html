概述

联发科技LinkIt™ ONE 是一个穿戴式与物联网（IoT）设备开发平台，提供近似于 Arduino 的硬件和 API ， 让您针对您的创意量身打造设备原型。

功能特色

  • 全方位通信功能：基于联发科技 MT2502A系统单芯片，支持各种通信与多媒体功能，支持 GSM、GPRS、蓝牙 2.1 和 4.0 、SD 卡、MP3/AAC 音频，并集成Wi-Fi 和卫星定位系统

  • 与 Seeed Studio 合作的 LinkIt ONE 开发板，拥有与 Arduino UNO 类似的引脚，让您连接许多不同的周边组件

  • LinkIt ONE SDK提供开发者熟悉的 Arduino 环境和简单易用的工具，让初学者能轻松进行开发

开发板

LinkIt™ ONE

硬件开发套件内含有 LinkIt ONE 开发板、Wi-Fi与蓝牙天线，GSM (2G)天线、GPS/GLONASS天线，以及锂电池，您能很快的上手，实现您的构想。


您能替设备加上额外的 Grove 感应器或者任何与开发板兼容的周边配件。


与 Seeed Studio 合作设计，此开发套件有您入门所需要的一切。

Key features

  • 支持 GSM, GPRS, Wi-Fi, 蓝牙（包含2.1 SPP 和 4.0 GATT 协议），卫星定位 (GPS,GLONASS 和北斗)、音频输出，和SD 卡接口

  • 与 Arduino UNO 相似的接脚排列，包含数字与模拟输出入、PWM、I2C、SPI、UART 与电源供应

  • 开源硬件参考设计，提供原理图，布局和接脚细节

AWS IoT Starter Kit

此工具套件提供 LinkIt ONE 开发板，以及由 Seeed Studio 提供的电池和天线，还有多种 Grove 感应器。


搭配 LinkIt ONE SDK，以及mbed TLS 程序库 和 AWS IoT软件开发套件，您可以快速地实现穿戴式与物联网概念，并利用 AWS 云服务创建物联网设备。

芯片

MT2502A

LinkIt ONE 开发版的核心为联发科技 MT2502A。


联发科技 MT2502 是一个功能丰富而强大的系统单芯片，具备高端 GSM/GPRS(2G) 的通信能力，并且还集成了完全兼容于 Blueooth v4.0 标准的 Bluetooth 硬件。


MT2502 还可以搭配联发科技的 Wi-Fi(MT5931) 与 卫星定位(MT3332) 芯片组，为穿戴式与物联网设备提供高性能、低功耗的解决方案。

SDK

LinkIt ONE SDK (for Arduino) 是 Arduino IDE 的扩展，此套件提供可靠且有弹性的方法来协助您在 LinkIt ONE 开发板上面构思与验证您的穿戴式与物联网设备构想。

功能特色

  • 编译 Arduino Sketches

  • 支持 Microsoft Windows XP, Vista, 7, 8 and 10，以及 Apple Mac 10.09 and 10.10

  • 提供 LinkIt ONE API，支持核心的 Arduino 功能，以及 LinkIt ONE 专有的通信功能

  • 通话与 GPRS 数据发送 API

  • 卫星定位 (GPS, GLONASS and BeiDou) API

  • Bluetooth SPP API

  • BLE GATT API

  • Wi-Fi API，并提供 TCP and UDP 连接功能
