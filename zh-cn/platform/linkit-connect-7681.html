LinkIt™ Connect 7681

API references 开发者指南
讨论区 FAQ

概述

随着具有无线功能的家电用品日益增加的趋势，对于许多智能家电设备的开发者来说，联发科技 MT7681 所提供的从远程控制设备的能力 – 例如开启枱灯，调整空调温度设置或解除大门锁定等等，正是此平台的最佳用途与优势。


针对物联网设备原型设计，我们提供了 LinkIt Connect 7681 开发板。该开发板由一个 MT7681 模块为核心，加上 micro-USB 接口并提供MT7681芯片的所有I / O接口引脚。 这能使您快速的连接外部硬件周边来创建设备原型。 MT7681 模块的大小只有15x18mm，此设计让它能够轻易的集成至最终物联网设备的PCB上。

Key features

  • 支持 Wi-Fi STA 模式: 802.11b/g/n

  • 支持 Wi-Fi AP 模式: 802.11b/g

HDK

LinkIt Connect 7681

此 LinkIt Connect 7681 开发板是与 Seeed Studio 协同设计的。它提供您五个 GPIO 接脚和一个UART 端口，并各有专属的 LED，让您快速连接周边硬件和外部电路。本开发板基于联发科技 MT7681，是具备高集成度的无线 SoC，并带有 TCP/ IP 协定堆叠。此开发板上含有和 AcSiP 协同设计的 LinkIt Connect 7681 模块，这个模块只有15×18 毫米，也可分别由 Seeed Studio 和AcSiP 批发商获得，并直接集成至您的最终产品的 PCB 上。

Key features

  • 五个 GPIO 接脚和一个 UART 端口，和相应的 LED

  • 集成式天线

  • USB 电源 (Micro USB 接口)

芯片

MT7681

支持 TCP/ IP 协定的嵌入式 Wi-Fi 物联网系统单芯片。


联发科技 MT7681 是个高集成的无线网络系统级芯片（SoC），并拥有物联网设备所需的嵌入式TCP/ IP协定堆叠。IoT 设备经由 MT7681 将可以连接到其他智能设备或云端应用和服务。


在 STA 模式时，MT7681 连接至无线 AP，之后可将与网站服务或云端服务器进行通信。一个典型的用途是，用户可透过智能家居的网站来远程控制家里的冷气机。

SDK

联发科技 LinkIt™ Connect 7681 SDK ，是基于 Andes 平台的软件工具开发包。它能让您创建支持 Wi-Fi 网络的物联网设备固件。该 SDK 提供固件更新工具，以及LinkIt Connect 7681 开发板软件上传工具。

功能特点

  • LinkIt Connect 7681 API 库包含Smart Connection 和空中固件更新（FOTA）

  • 基于 C 语言

  • 基于 Andes 平台的命令行编译器

  • 提供 Wi-Fi 与互联网 API、周边接口 API，以及安全加解密 API

  • 支持 Wi-Fi AP 和 STA 模式. 支持 Wi-Fi 安全协定，包含WEP, WPA 和 WPA2-PSK。网络协定则包含 TCP/IP, DHCP, SNTP 和 DNS client

  • 支持微软Windows（需安装 Cygwin 环境）和 Ubuntu Linux
