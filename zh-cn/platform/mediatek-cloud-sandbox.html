MediaTek Cloud Sandbox

概述

在设计与开发可穿戴和物联网设备原型时，您可能希望能收集，排序和视觉化开发原型所收集到的数据。您或许也想透过远程控制您的设备原型，并且提供这些功能给一同协作的开发者与测试人员。

现在您无需花费过多的时间与金钱来寻找或购买这类云端服务，MediaTek Cloud Sandbox（MCS）为您提供免费的服务，您可以使用它快速的打造出您所构想的云端服务的原型。


您可以在我们提供的强大的网页仪表板介面上，看到使用我们提供的 RESTful API 从装置搜集回来的图形化的资料。之后，您可以通过从仪表板发出命令控制您的穿戴式和物联网装置。此外，我们提供附属的手机应用程式让您能查看搜集到的数据，并从任何地方控制您的装置。


MCS 目前提供服务给全球的会员，并且在中国内部架设有独立的资料中心，用以提供最佳的连线品质与服务。

重点功能

  • 定义专属于您的穿戴式和物联网产品原型以及个人化装置。

  • 定义各种类型的资料点。例如：GPS 座标，温度，湿度。

  • 在同个产品原型下建立多个装置。

  • 透过 Restful API 随时在装置和 MCS 平台间推播各式资料点。

  • 远端控制您的装置。例如：远程开关设定。

  • 图形化的资料点介面。

  • 当资料点超过特定触发条件预设范围时，发送通知。

  • 使用专属的手机应用程式远端控制装置。

  • 建立产品原型，装置，和搜集来的资料报表。

  • 空中（Over the air）韧体更新。

  • 使用者权限管理。

  • 完整的教学文件。
