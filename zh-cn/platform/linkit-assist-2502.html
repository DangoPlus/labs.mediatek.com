LinkIt™ Assist 2502

API 指南
技术资源
讨论区 FAQ

概述

联发科技LinkIt™ Assist 2502 让开发者在专业的开发环境下进行穿戴式与物联网（IoT）设备的硬件和软件原型开发。


本开发平台是基于现今最小的穿戴式系统级芯片组 (SoC), 联发科技 MT2502 （Aster）。此SOC 也能够和联发科技节能的 Wi-F i与 GNSS 芯片组一同运作，而这意味着您现在就可以很轻松的创建设备并与其他智能设备或云端服务连接。

LLinkIt Assist 2502开发平台是由 MT2502 (Aster) 和配套 GNSS 与 Wi-Fi 芯片组, 还有触摸皮肤 LCM CTP，Memory LCD 和OLED 屏幕（可更换）所构成的 LinkIt Assist 2502 开发平台，它的固件是以 RTOS 为核心。在核心之上运行着各种驱动程序、仲介软件与通信协定。Runtime运行环境提供了以C语言为基础的API来访问系统功能，让您使用LinkIt Assist 2502 软件开发工具包 SDK 2.0 创建应用。

功能特点

  • 基于 C 语言的 API，让您访问和控制 MT2502 SoC 的硬件功能和周边硬件

  • 提供 API 让您更新软件和固件，还有能运行在线固件更新的可能性。

  • 综合各种通信、多媒体与用户接口，支持：

  • GSM, GPRS and 蓝牙 2.1 和 4.0 (由 MT2502A 支持)
  • 无线网络与卫星定位 (由配套芯片组支持)
  • MP3/ AAC 音频播放
  • JPEG 解码, 矢量字体 (由 Etrump 提供)，以及其他多媒体功能

HDK

LinkIt Assist 2502

这块与 Seeed Studio 合作设计的开源硬件开发板，包含了LinkIt Assist 2502 模块和多功能的 Wi-Fi与 GNSS模块。它还提供了可替换的触摸显示屏幕，以及可连接外部硬件的完整接脚输出入接口。


只需要一块板子，就可得到全面的通信和定位功能。

功能特点

  • 支持GSM, GPRS, Wi-Fi, 蓝牙（ 2.1 SPP和 4.0 GATT）, GNSS (GPS, GLONASS 和 北斗) 和音频输出。

  • 输出接脚，包含数字I/O、模拟I/O、PWM、I2C、SPI、UART与电源供应。

  • 可替换的显示器（包含一正方形LCM CTP）

  • 开源开发板硬件参考设计，包含原理图、布局图与接脚细节

芯片

MT2502A

LinkIt Assist 2502 开发版的核心为联发科技 MT2502A。


联发科技 MT2502 是一个功能丰富而强大的系统单芯片，具备高端 GSM/GPRS(2G) 的通信能力，并且还集成了完全兼容于 Blueooth v4.0 标准的 Bluetooth 硬件。


MT2502 还可以搭配联发科技的 Wi-Fi(MT5931) 与 卫星定位(MT3332) 芯片组，为穿戴式与物联网设备提供高性能、低功耗的解决方案。

了解更多

SDK

LinkIt Assist 2502 SDK 2.0 提供一个 Eclipse IDE (with CDT) 的插件，让您进行软件开发。该软件开发包包含了固件更新工具和 LinkIt Assist 2502 开发板软件上传工具。

功能特点

  • C 语言 API 开发环境

  • 编译专案至LinkIt Assist 2502 运行文档 (.vxp)

  • LinkIt Assist 2502 API 函式库提供以下通信功能

  • TCP 连接
  • HTTPS
  • Bluetooth 4.0 GATT 与其他功能

  • 支持矢量字库（由驿创提供），JPEG解码，影像，并显示在屏幕上

  • 与 Eclipse IDE (Indigo) CDT 兼容（8.0.2 版或更新）

  • 支持微软Windows XP，Vista，7 和 8

注意

LinkIt Assist 2502 SDK 亦可用于LinkIt ONE 开发板，唯部分功能可能受到硬件限制。请参阅LinkIt Assist 2502 开发者指南了解详情。
