开发平台

LinkIt™ 开发平台

LinkIt 7697

具备 Wi-Fi 与低功耗蓝牙连接能力，可使用 Arduino IDE 或基于 FreeRTOS 的 LinkIt SDK 开发。

LinkIt Smart 7688

打造有无线网路功能的智慧家庭与办公室装置原型。运行 OpenWrt Linux，可支援 Python, Node.js 与 C。

LinkIt ONE

用 Arduino IDE 在这张具备多样连线功能的开发版实验您的创意构想。

舊有開發平台

LinkIt Connect 7681

已停止销售 - 内容维护至 2018 八月

打造具备 Wi-Fi 功能，可连线至智慧型手机与云端服务的智能插座、灯泡等智能家电。

LinkIt Assist 2502

已停止销售 - 内容维护至 2018 八月

制作精巧而具备可穿戴尺寸规格的概念雏形。提供 C 语言开发者功能丰富的 Eclipse 开发环境。

Helio™ 平台

MediaTek X20 Development Board

具备十核心 Helio X20 处理器的先进 96Boards 开发板。

云端服务

MediaTek Cloud Sandbox

MediaTek Cloud Sandbox (MCS) 是一个云端资料服务平台，让您能够连结物联网装置。
