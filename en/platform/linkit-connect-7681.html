Overview

The increasing trend towards enabling smart home that involves connecting every imaginable electrical or electronic device is spreading like wildfire. For many of you applications developers who simply want to add the ability to remotely control a device, such as turning on a table lamp, adjusting the temperature setting of an air-conditioner or unlocking a door. LinkIt Connect 7681 is what you’ve been waiting for.


LinkIt Connect 7681 is designed for IoT device prototyping. The development board consists of a LinkIt Connect 7681 module, a micro-USB port and pins for each of the I/O interfaces of the MT7681 chip. This enables you to quickly connect external hardware and peripherals to create device prototypes. The LinkIt Connect 7681 module, which measures just 15x18mm, is designed to easily mount on a PCB as part of production versions of an IoT device.

Key features

  • Wi-Fi Station mode: 802.11b/g/n

  • Wi-Fi Access Point mode: 802.11b/g

HDK

LinkIt Connect 7681

The LinkIt Connect 7681 development board, co-designed with Seeed Studio, provides you with easy access to the five GPIO pins and one UART port — each with an LED — of the MediaTek MT7681 chip, to quickly connect peripheral controllers and electronic circuits. The MediaTek MT7681 — compact Wi-Fi SoC with embedded TCP/IP stack — is provided on the LinkIt Connect 7681 module, co-designed with AcSiP.

Key features

  • Access to the five GPIO pins and one UART port provided by MT7681, with attached LEDs

  • Integrated Wi-Fi antenna

  • Power over USB (Micro USB port)

Chipsets

MT7681

Wi-Fi System-on-Chip (SoC) for IoT devices with embedded TCP/IP stack.


MediaTek MT7681 is a compact Wi-Fi System-on-Chip (SoC) for IoT devices with embedded TCP/IP stack. By adding the MT7681 to an IoT device, it can connect to other smart devices or to cloud applications and services. Connectivity on the MT7681 is achieved using Wi-Fi in either Wi-Fi station or access point (AP) mode.


In Wi-Fi station mode, MT7681 connects to a wireless AP and can then communicate with web services or cloud servers. A typical use of this option would be to enable a user to control the heating in their home from a home automation website.

SDK

The MediaTek LinkIt™ Connect 7681 SDK, based on the Andes development kit, enables you to create firmware to control an IoT device in response to instructions received wirelessly over a Wi-Fi connection. The SDK also provides tools to update firmware on and upload software to the LinkIt Connect 7681 development board.

Key features

  • Libraries for the MediaTek LinkIt Connect 7681 API, including Smart Connection and FOTA firmware updates

  • C-Language

  • Command line compiler, based on Andes Development Kit

  • Security, Control and Interface API, Internet API with Wi-Fi functionality

  • Supports Wi-Fi station and access point (AP) modes; Wi-Fi security protocols including WEP, WPA and WPA2-PSK and networking protocols including TCP/IP, DHCP, SNTP and DNS client are provided

  • Supports Microsoft Windows (requires Cygwin cross-compiler tool) and Ubuntu Linux
