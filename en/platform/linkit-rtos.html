LinkIt™ Development Platform for RTOS

Overview

The MediaTek LinkIt™ Development Platform for RTOS supports a range of chipsets, including the MediaTek MT7687F, MT7697 and MT7697D for IoT devices and MT2523D and MT2523G for Wearable. This development platform offers the convenience of a common toolset and common APIs implemented over popular FreeRTOS. With this you can achieve economies across a full range of consumer and business IoT devices.

Connectivity SoC

A family of power-efficient system-on-a-chip (SoC) products that combine various connectivity technologies with a microcontroller, including:


  • Wi-Fi (MT7687 family)
  • Wi-Fi and Bluetooth LE (MT7697 family)
  • GNSS, Bluetooth EDR and LE (MT2523 family)

Popular architectures

Based on the popular ARM Cortex-M4 hardware architecture and FreeRTOS, LinkIt development platform for RTOS enables you to take advantage of the many third-party resources in these ecosystems.

Rich software features

The LinkIt SDK v4 includes a wide range of software modules that will help speed up your development. These modules support features such as TCP/IP, SSL/TLS, MQTT, Bluetooth EDR/LE profiles, sensor subsystem, battery management and FOTA.

HDKs

LinkIt 7697

The LinkIt 7697 is an Internet of Things (IoT) applications development board based on MediaTek MT7697 SoC. The MT7697 chip is based on ARM Cortex-M4 processor with FPU and features 802.11b/g/n Wi-Fi and Bluetooth 4.2 Low Energy connectivity.

Features

  • ARM Cortex-M4 based MT7697 SoC @ 192MHz

  • 802.11b/g/n wireless connectivity

  • Bluetooth 4.2 Low Energy

  • 352 KB of RAM

  • 4 MB external Flash memory

  • Micro-USB port for power supply

  • Virtual COM port through onboard CP2102N USB-to-UART chipset

  • Two buttons for system reset and external interrupt

  • Peripheral interfaces including GPIO, UART, I2C, SPI, PWM, EINT, ADC, IrDA, I2S

  • Pin headers for SWD debugging

  • LinkIt SDK GCC, IAR Embedded Workbench, and Keil uVision IDE (Get Started Guide)

  • Additional add-on Package for Arduino IDE support

  • Size: 48 x 26 mm (1.89 x 1.02 inches)

LinkIt 7687 HDK by SAC

The LinkIt 7687 HDK by SAC is an Internet of Things (IoT) development board with which to design, prototype, evaluate and implement IoT projects. It is based on MT7687F, with ARM Cortex-M4 processor with FPU in a QFN68 package that provides 802.11b/g/n Wi-Fi connectivity. The HDK offers rich peripheral connectivity features, the ability to communicate with cloud services, and real-time control of peripheral hardware.

The LinkIt SDK v4 includes libraries for use with the LinkIt 7687 HDK to integrate a variety of peripherals and create a new class of highly connected applications.

Features

  • ARM Cortex-M4 based MT7687F SoC

  • 802.11b/g/n wireless connectivity through on-board antenna

  • U.FL connector for conducted testing

  • Micro-USB port for power supply, virtual COM port, and CMSIS-DAP interface

  • External VIN 1.8~3.2V allows 2xAA batteries as power source

  • Three buttons for system reset, Real Time Clock (RTC) interrupt and external interrupt

  • Headers for power consumption (current) measurement

  • Arduino Uno (Revision 3) pin compatibility plus an eight pin extension connector

  • Size: 60.5 x 108.5mm (2.4 x 4.27 inches)

LinkIt 2523 HDK by SAC

The LinkIt 2523 HDK by SAC is a fully functional development board for wearable application prototypes powered by MediaTek MT2523G, an ARM Cortex-M4 processor with FPU that provides Bluetooth EDR/LE and GNSS. The board offers a touch display and extension connectors to battery and sensors.

Features

  • ARM Cortex-M4 based MT2523G SoC

  • Bluetooth EDR/LE and GNSS support

  • On-board antenna and an SMA connector for external antenna

  • Micro-USB ports for power supply, virtual COM port, and CMSIS-DAP interface

  • 320 x 320 pixel touch display

  • Speaker and headset jack

  • 4GB eMMC flash storage

  • SD card slot

  • Extension connectors for battery, sensors and camera

  • Headers for power consumption (current) measurement

  • Size: 70 x 215 mm (2.76 x 8.46 inches)

SDK

The MediaTek LinkIt™ Software Development Kit (SDK) v4 provides the software and tools for the development of applications in a three layer architecture, consisting of BSP, Middleware, and Application layer.


The BSP(board support package) layer includes drivers for on-chip hardware integrated on top of FreeRTOS, and provides a common hardware abstraction layer API across different chipsets. It also includes drivers for on-board hardware such as displays and touch panels.


The middleware layer provides a framework of essential modules for application development, such as Wi-Fi and Bluetooth stacks, lightweight IP (lwIP), internet protocols, battery management, sensor subsystem, Firmware update Over-The-Air (FOTA), and more.


The application layer provides useful example projects, reference designs, and demo projects for developers to quickly adopt into their own projects.


You’ve the option to develop with the free GCC toolchain, and professional IDEs including Keil μVision and IAR Embedded Workbench.


Supported development environments

  • Plug-in for ARM Keil μVision on Windows
  • IAR Embedded Workbench on Windows
  • GCC on Linux or Windows (using MinGW)
